## How to Run
1. Run:
```bash
# Clone this repository
git clone https://github.com/Windows87/programming-language-akinator/
# Go into repository
cd programming-language-akinator
# Install dependencies
pip install -r requeriments.txt
# Start
python run.py
```
2. Go to http://localhost:5000
